# 网页版战术射击（瓦罗兰特玩法致敬）设计文档

日期：2026-07-16

## 目标
纯前端网页 FPS，尽量还原无畏契约的核心玩法节奏。所有美术/音频程序化生成，不使用任何官方版权素材。

## 技术
- 纯静态 HTML + ES Modules，Three.js 经 CDN import map 引入，无构建步骤。
- 碰撞/弹道全部用解析 AABB/球体射线检测；渲染用 InstancedMesh。

## 模式
- 5v5 炸弹模式（玩家 + 4 AI 队友 vs 5 AI），先赢 13 回合。
- 购买 30s / 回合 100s / 下包后 45s 爆炸；安放 4s、拆除 7s（半拆记忆）。
- 第 12 回合后换边；12-12 突然死亡决胜局。

## 经济（贴近原版）
起始 800；击杀 +200；下包全队 +300；胜 +3000；连败 1900/2400/2900；上限 9000。
轻甲 400（25），重甲 1000（50）；护甲吸收 66% 伤害。

## 武器
Classic/Ghost/Sheriff/Frenzy/Stinger/Spectre/Bucky/Judge/Bulldog/Guardian/Phantom/Vandal/Marshal/Operator/Ares/Odin/近战刀。
含：伤害分部位（头/身/腿）、距离衰减、射速、弹匣、换弹、开镜、首发精准、移动扩散、后坐力爬升。

## 特工（自制命名，玩法致敬）
- 风影（决斗者）：侧风烟 / 上升气流 / 疾风突进（免费）｜大招：锋刃风暴（7 点）
- 天穹（控场者）：燃烧弹 / 兴奋剂 / 空降烟幕 x3｜大招：轨道打击（8 点）
- 圣愈（哨卫）：屏障之墙 / 缓速球 / 治愈之光（免费）｜大招：复生（8 点）

## 地图
自制双点位图（Ascent 式三通道）：A 主道 / 中路 / B 主道 + A/B 小道 + 后场连接。
开放区域声明 + 栅格化生成墙体 AABB；waypoint 图自动连边（LOS + 距离）供 AI 寻路。

## AI
状态机：等待/进攻推点/驻守/交火/安放/拆除/回防。视线检测（墙体+烟雾遮挡）、反应延迟、距离相关瞄准误差、点射节奏、死亡广播回防。

## 模块
state(全局G) / config / utils / audio(WebAudio程序化) / map / effects / combat(伤害与弹道) /
player(控制器) / abilities / bots / game(回合状态机) / hud(DOM界面) / main。
模块间回调经 G.hooks 解耦避免循环依赖。
